# Banner Bot Handoff

## Что это

Telegram-бот для генерации серии баннеров по документу с опорой на одну референс-картинку стиля.

## Актуальный сценарий работы

1. Пользователь отправляет `/start`
2. Отправляет ссылку на документ или `.docx` файлом
3. Бот разбирает заголовки
4. Бот просит одну референс-картинку
5. Бот генерирует серию баннеров в стиле референса
6. Если нужно, конкретный баннер можно переделать через кнопку `Переделать`

## Как запустить на Linux

1. Установить Node.js 20+ (рекомендуется Node 22)
2. Клонировать репозиторий
3. Создать `.env` на основе `.env.example`
4. Вставить рабочие токены
5. На Linux-сервере обязательно выполнить:

```bash
npm install
npm start
```

## Важно

- В репозитории нет `.env`, `node_modules` и `output`
- Это нормально: на Linux-сервере зависимости обязательно нужно ставить локально через `npm install`
- Если процесс перезапустить, активные сессии бота сбросятся, потому что состояние хранится в памяти

## Команды

- Старт: `npm start`
- Microsoft device login helper: `npm run microsoft:device-login`
